# Web Scrapping: eBay Consumer Electronics

This project extracts product names and prices from eBay's Consumer Electronics category, then saves the product data to a CSV file.

## Features
- Scrapes product names and prices.
- Saves the extracted data in a CSV file.
- Appends new product data to the existing CSV file, ensuring continuous data collection

## Tools and Technologies
- **Python** as the main programming language
- **Beautiful Soup & httpx**: For HTML parsing and data collection
    - **HTTPX** is suitable for modern web applications that need asynchronous HTTP requests.
    - **Beautiful Soup** is a web scraping library used to parse and extract data from HTML.
